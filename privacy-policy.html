Politique de confidentialité – Color Takeover

Dernière mise à jour : mars 2026

1. Introduction

Color Takeover est un jeu mobile développé par un développeur indépendant. Cette politique explique comment nous traitons vos données.

2. Données collectées

Color Takeover ne collecte aucune donnée personnelle. Le jeu fonctionne entièrement hors ligne. Aucune information n'est envoyée à des serveurs distants.

Les seules données stockées sont :

  • Votre progression dans le jeu (niveaux complétés, scores)
  • Vos préférences (langue, volume, skin sélectionné)
  • Vos statistiques de jeu (parties jouées, victoires)

Ces données sont stockées uniquement sur votre appareil (stockage local) et ne nous sont jamais transmises.

3. Publicités

Color Takeover ne contient aucune publicité.

4. Achats intégrés

Color Takeover ne propose aucun achat intégré.

5. Services tiers

Le jeu n'utilise aucun service tiers (pas d'analytics, pas de réseaux sociaux, pas de SDK publicitaire).

6. Enfants

Color Takeover ne collecte pas de données sur les enfants ni sur qui que ce soit. Le jeu est adapté à tous les âges.
